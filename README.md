# Onyx

<img width="150" src="assets/onyx.png">

## Descripción

Onyx es una aplicación que permite a sus usuarios crear y gestionar grupos de tareas.

## Arquitectura

Onyx está compuesto por 3 componentes principales:

- **API (Backend)**: API REST desarrollada con Spring Boot
- **Escritorio**: Cliente de escritorio desarrollado con JavaFX
- **Móvil**:  Aplicación Android desarrollada con Kotlin

## Tecnologías

### API
- JDK 21
- Maven
- Spring Boot 3.5.9
- MariaDB

### Escritorio
- BellSoft Liberica 21
- Maven

### Móvil
- Kotlin
- Android SDK 36
- Gradle
