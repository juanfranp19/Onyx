# Onyx

Proyecto llevado a cabo durante 2º DAM.   
Aplicación para gestionar tareas organizadas por grupos. Indicando fecha de vencimiento y si ha sido completada.

## Tecnologías

### Backend

- **API REST**:
   - Spring Boot
   - Java
   - Maven
   - JDK 21
   - MariaDB

### Frontend

- **Aplicación de escritorio**: 
   - JavaFX
   - Java
   - Maven
   - BellSoft Liberica 21

- **Aplicación móvil**:
   - Android
   - Kotlin
   - Gradle
   - Android SDK 36
